# 💰 Finance Tracker
## 🚀 Live Demo
Check out the live application here: [https://finance-tracker-frontend-8c9v.onrender.com](https://finance-tracker-frontend-8c9v.onrender.com)

A full-stack personal finance management application built with the MERN stack (MongoDB, Express, React, and Node.js). Track expenses, monitor spending habits, and gain insights into your financial activity through an intuitive dashboard.

---

✨ Features

- 🔐 Secure JWT Authentication
- 📊 Financial Dashboard
- 💸 Expense Tracking & Management
- 📈 Spending Analytics & Charts
- 📱 Responsive Design
- ⚡ Fast and Modern User Interface

---

🛠 Tech Stack

Frontend

- React
- Tailwind CSS
- Chart.js
- Axios 
- React Icons

Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt

---

🚀 Installation

Prerequisites

- Node.js (v16 or higher)
- MongoDB Atlas or Local MongoDB
- Git

Clone Repository

git clone https://github.com/your-username/finance-tracker.git
cd finance-tracker

Backend Setup

cd backend
npm install

cp .env.example .env
# Add your MongoDB URI and JWT secret

npm run dev

Frontend Setup

cd frontend
npm install
npm run dev
